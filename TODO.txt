Task 1)
Integrate the "Find by Merge Request" functionality into the "Filter by branch name" box on the dashboard. If a MR URL is entered, if that MR is already part of a tracked merge group then filter the display to that merge group. If not, show a button in the display area to "Open MR as Merge Group" that will do that the "Find" button in the "Find by Merge Request" box does now. Change the label in the filter box to "Filter by branch name or MR URL". Make sure that MR URLs with extra text at the end, such as "http://localhost:8081/test-group/primary-1/-/merge_requests/4/diffs" are still recognized as valid MR URLs.

Remove the "Find by Merge Request" box from the top bar.

As a QoL feature, add a paste icon/button into the filter box to paste contents from the clipboard.

Task 2)
The dashboard wastes screen space by being a single column display. Make the dashboard take up the full width of the browser window and display the merge groups in a multi-column layout. Make sure to still stick to the time groups (such as "Today", "Yesterday", etc). Within each time group, order the merge groups by last updated first, from left to right then top to bottom (i.e. the logical chronological order). The number of columns should be determined by the width of the browser window, with a minimum of 2 columns and a maximum of 4 columns. The merge group cards should be resized to fit the column layout while maintaining readability and usability.

Task 3)
In the merge group details page, next to each branch name label in the cards, add a "Copy" icon/button to copy the branch name into the clipboard.

Task 4)
In the merge group details page, the "Add Merge Request..." button should be blue and its icon should be a plus sign (or a similar traditional "add" icon).

Task 5)
In the merge group details page, the "In my Merge Groups" / "Add to my Merge Groups" toggle, this should be a button rather than a toggle label. When not in my merge groups, the button should say "Track Merge Group" and be blue with a plus sign icon. The tooltip should say "Track this merge group in my dashboard".

When in my merge groups, the button should say "Untrack Merge Group", be grey with a minus sign icon. The tooltip should say "Remove this merge group from my dashboard".
